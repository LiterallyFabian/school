About the game

The UI

In stellaris there's much content. many different resources and research and and many gameplay features. when you start playing the game it might be hard understand it but after a while it's probably one of the better game UI for game.

Story

The game have much story to it as well. There are many different events and many different creatures in the game with there own story. I have played for more that 100 hours and i still find new things. Some story based elements are locked behind dlc “in game purchase” so may need to pay extra if you want the full expressions. and if you like strategy and story based elements i recommend it.

Workshop

The game supports modding which will enable you to download and enhance your experience or just give you a fresh look on the game.
